Section

        

          Обёртка для layout-блоков каталога: title, описание, body. Примитивы выше — в .panel.panel--content (dots + title).
          Модификатор .section--sticky — sticky-панель на desktop (≥769px).
        

        

          

            

              
Длинный контент слева: прокрутите страницу, чтобы проверить sticky-поведение панели справа.

              
Каждый примитив на этой странице живёт в panel--content. Section — для layout-примитивов и sticky-демо.

              
На mobile (≤768px) модификатор не активен — колонки складываются в одну, панель идёт после контента.

              
Sticky работает только внутри scroll-контейнера страницы: top: var(--space-6), высота ограничена viewport минус отступы.

              
Если содержимое панели выше лимита, внутри неё появляется собственный scroll (overflow: auto).

              
Пример использования — двухколоночный layout: основной текст и боковая панель с действиями или сводкой.

              
Комбинируйте с .grid.grid--2 для desktop/mobilе адаптации без дополнительного CSS.

              
Дополнительный абзац, чтобы высота левой колонки гарантированно превышала viewport и sticky было заметно при прокрутке.
